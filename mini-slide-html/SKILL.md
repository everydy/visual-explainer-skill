# mini-slide-html

`mini-slide.html` 하나를 책임지는 하위 스킬이다.

## 검색 호출어

영어 호출어 `mini slide html`이 보이면 이 스킬을 우선한다.

- `mini slide html`
- `mini-slide-html`
- `mini-slide.html`
- `미니 슬라이드 HTML`
- `슬라이드 HTML`

## 역할

내용을 작은 발표 자료처럼 장면별로 넘기며 이해하게 만든다.

## 입력

- 핵심 메시지
- 사용자가 따라가야 할 이야기 순서
- 각 장면의 제목과 설명
- 저장 파일명: `mini-slide.html`

## 디자인 판단

- 몇 장이 가장 적당한지 정한다. 기본은 5-8장이다.
- 각 장면을 문제, 배경, 구조, 예시, 결론, 다음 행동 중 무엇으로 둘지 정한다.
- 슬라이드 이동을 버튼, 키보드, 탭 중 무엇으로 할지 정한다.

## 산출물

- `mini-slide.html`
- 슬라이드 이동 UI, 현재 장면 표시, 짧은 문장, 필요하면 작은 도식을 포함한다.

## 검증

- 첫 장에서 무엇을 설명하는지 바로 알 수 있다.
- 마지막 장에서 핵심 결론과 다음 행동이 보인다.
- 각 장면은 너무 많은 내용을 담지 않는다.
- 버튼이나 키보드로 장면 이동이 가능하다.

## 금지

- 긴 문서를 슬라이드처럼 잘라놓기만 하지 않는다.
- 한 장에 너무 많은 글을 넣지 않는다.
